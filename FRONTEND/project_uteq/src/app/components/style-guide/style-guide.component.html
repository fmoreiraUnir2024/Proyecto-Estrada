Writing Style Guide

Recommendations for maintaining a consistent, professional writing style.

Use Third Person

Refer to the user as "the user" or "they" instead of "you".

Incorrect: You can access your account by clicking the button below.

Correct: The user can access their account by clicking the button below.

Maintain a Formal Tone

Avoid contractions, slang, and overly casual language.

Incorrect: Hey there, check out this cool feature!

Correct: Please note that this feature is available for your use.

Formatting Guidelines

Recommendations for maintaining a consistent, readable format.

Use Headings Effectively

Organize content with clear, descriptive headings.

Correct Heading Structure

1. Introduction
2. Key Features
2.1. Feature A
2.2. Feature B
3. Conclusion

Maintain Consistent Formatting

Use the same font, size, and spacing throughout the document.

Incorrect: Inconsistent font sizes and spacing can make the document appear disorganized.

Correct: Maintaining consistent font, size, and spacing throughout the document creates a clean, professional appearance.
